# digital-literacy-project
A comprehensive Digital Literacy Portfolio developed as part of CSE0001 (VIT Bhopal), covering infographic design, digital portfolio creation, coding platforms, professional email etiquette, and cybercrime awareness.

# Project Overview

This project was completed as part of the Digital Literacy course at VIT Bhopal.  
It demonstrates essential digital skills such as creating visual content, building a professional online presence, using coding and collaboration platforms, practicing email etiquette, and understanding cybercrime awareness.

# Tasks Completed

# Task 1 – Digital Literacy Infographic  
Created an infographic explaining digital literacy, safe internet practices, and professional online presence using Canva.

# Task 2 – Digital Portfolio  
Developed profiles on GitHub, LinkedIn, and Kaggle to establish a professional online presence.

# Task 3 – Platforms  
Solved a beginner problem on HackerRank and created a Google Form quiz for digital literacy awareness.

 *Google Form Link:* https://docs.google.com/forms/d/e/1FAIpQLScPuZo_VG9Ork2o4P9IbwTHfb3fSAXcznaVJCLPByPDVZ5BdA/viewform?usp=header

# Task 4 – Email Etiquette  
Drafted professional emails and created a checklist for responsible social media usage.

# Task 5 – Cybercrime Awareness  
Prepared a case study on phishing and created a prevention checklist for online safety.
